# About
Flutter appにFlutter library(SDK)が搭載可能かを検証するリポジトリ

- app : Flutter app
    - flutter stable

- lib : Flutter Library(SDK)
    - flutter 3.19.3

# How to

- Enviroment

1. install fvm
1. fvm install (3.19.3 & stable)
1. launch

# confirmation item

1. runZonedGuarded
    アプリとLibraryでそれぞれrunZonedGuardedを設けていた場合の挙動と回避策の模索
