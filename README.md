# About
Flutter appにFlutter module(SDK)が搭載可能かを検証するリポジトリ

- app : Flutter app
    - flutter stable

- sdk : Flutter module(SDK)
    - flutter 3.19.3

# How to

- Enviroment

1. install fvm
1. fvm install (3.19.3 & stable)
1. launch
    (for VScode)
    launch and please select app/sdk
    (other)
    cd app && flutter run
    cd sdk && flutter run
